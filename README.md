### Kurzbeschreibung

Quantum Client ist eine leistungsstarke Browser-Erweiterung für das Spiel, die zahlreiche Module und Werkzeuge bereitstellt, um das Spielerlebnis zu erweitern und anzupassen. Die Erweiterung bietet Automatisierung, Analyse-, UI- und Audio-Funktionen sowie experimentelle KI-Features in einer konsistenten, konfigurierbaren Oberfläche.

### Features (Kurz)

* Modular: Module separat aktivierbar / konfigurierbar
* Performance & Netzwerk (FPS/Ping-Graph, Network Inspector)
* Automatisierung (Autoclicker, Makro-Recorder, Automations-Regeln)
* UI & Visuals (Themer, Theme Changer, Scale, Splash Screen)
* Spieler-Analyse (ESP, 3D-Tracker, experimenteller AI Tracker)
* Bedienkomfort (Quantum Menu, Keybinds, Sounds, Adblock Hider)

### Installation (Kurz)

1. Repo klonen oder ZIP herunterladen.
2. `chrome://extensions` öffnen.
3. Entwicklermodus aktivieren.
4. **Entpackte Erweiterung laden** → Ordner mit `manifest.json` wählen.
5. Erweiterung ist aktiv.

> Hinweis: Einige Funktionen nutzen die Debugger-API; benötigte Berechtigungen sind in `manifest.json` vorausgefüllt.

### Installation (short)

1. Clone the repo or download ZIP.
2. Open `chrome://extensions`.
3. Enable Developer mode.
4. Click **Load unpacked** and select the folder containing `manifest.json`.
5. Extension is installed and active.
